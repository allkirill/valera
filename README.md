📸 Valera Photo Loader
Valera Photo Loader — это настольное приложение, предназначенное для автоматизации массовой загрузки изображений по URL-адресам, указанным в файлах Excel. Оно имеет удобный графический интерфейс, надежную обработку ошибок для сложных веб-сайтов и удобные параметры постобработки, такие как изменение размера, добавление водяных знаков и переименование.

✨ Функции
Интеграция с Excel: считывает URL-адреса непосредственно из выбранных столбцов и обновляет файл Excel с цветовой кодировкой отчетов о состоянии (зеленый = успех, желтый = частичный сбой, красный = сбой).

Интеллектуальная загрузка:
подменяет заголовки User-Agent и Referer, чтобы обойти базовые средства защиты от сбора данных.

Игнорирует ошибки проверки SSL (полезно для старых или неправильно настроенных серверов).
Обработка объединенных ячеек: автоматически обнаруживает объединенные ячейки в Excel и предлагает разъединить их, чтобы предотвратить ошибки обработки.
Обработка изображений:
Изменение размера изображений по высоте или ширине.
Обрезка до квадратной формы (центральная обрезка).
Применение водяного знака PNG.
Преобразование в jpg, png или webp.
Контроль качества:
Фильтрация файлов по минимальному размеру (например, отклонение файлов размером менее 5 КБ).
Режим «Отклонить при ошибке» перемещает проблемные загрузки в отдельную папку FAIL.
Настраиваемый индикатор выполнения: забавный персонаж «Валера», который перемещается по индикатору выполнения во время процесса загрузки.
Гибкое именование: возможность переименовывать файлы на основе идентификатора статьи (например, Article_01.jpg) или сохранять исходные имена файлов.

🚀 Начало работы
Предварительные требования
Python 3.8+
pip
Установка
Клонируйте репозиторий:
git clone https://github.com/allkirill/ValeraPhotoLoader.gitcd ValeraPhotoLoader
Установите необходимые зависимости:
pip install pyside6 openpyxl pillow requests
(Необязательно) Поместите изображение Valera.png в корневую папку, чтобы увидеть пользовательский символ индикатора выполнения и значок окна.
Использование
Запустите приложение:
python main.py
(Замените main.py на имя вашего скриптового файла)
Выберите файл Excel: Нажмите «Выбрать Excel», чтобы выбрать исходный файл. Если файл содержит объединенные ячейки, приложение предложит вам исправить их.
Выберите папку для сохранения: Выберите место, куда будут сохранены изображения.
Настройка параметров:
Установите столбцы «Статья» (например, A) и «URL-адреса» (например, B-P).

Настройте размер и выравнивание изображения.
Выберите формат вывода и фильтры качества.
Запуск: Нажмите кнопку СТАРТ. Кнопка превратится в индикатор выполнения, показывающий статус загрузки.

⚙️ Параметры конфигурации
Описание настроек
Столбец «Статья» Столбец, содержащий уникальный идентификатор имени папки и имени файла.
Столбцы URL-адресов Столбцы, содержащие ссылки на изображения. Поддерживаются диапазоны (например, B-P) или отдельные столбцы.
Выравнивание Логика изменения размера: масштабирует изображение в зависимости от целевой высоты или ширины.
Отклонить при ошибке Если отмечено, строки с ошибками загрузки или небольшими файлами перемещаются в папку FAIL.
Минимальный размер файла Файлы меньшего размера, чем этот порог, обрабатываются как ошибки.
Переименовать по статье Если отмечено: Article_1.jpg. Если не отмечено: сохраняет исходное имя файла из URL.
